# java-kanban

## Описание
java-kanban — это программа для управления задачами. 

## Возможности
- Создание задач, подзадач и эпиков
- Получение списка всех задач
- Получение списка подзадач определённого эпика
- Удаление всех задач
- Удаление задачи по идентификатору
- Обновление информации о задаче
- Получение информации по идентификатору задачи
